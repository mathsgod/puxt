# puxt

## Debug
Set the `DEBUG` environment variable to `true` to enable debug mode.
```env
DEBUG=true
```

### Exception format
Set the `DEBUG_EXCEPTION_FORMAT` environment variable to `json` to enable exception format.
```env
DEBUG_EXCEPTION_FORMAT=json
```
